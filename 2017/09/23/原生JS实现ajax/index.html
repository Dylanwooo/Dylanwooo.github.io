原生JS实现ajax

ajax作为一种无须刷新页面即可从服务器中得取数据的技术，成为做前端必须掌握的一个知识点，同时原生JS实现ajax也是各大公司常考的经典面试题。但是，使用的时候往往会缺少兼容性和其他用户体验方面的考虑，本文从这些方面重新认识一下ajax的用法。

1.XMLHttpRequest对象

IE是第一款引入XHR对象的浏览器。在IE5中，XHR对象是通过MSXML库中一个ActiveX对象实现的。因此，在IE中可能会遇到三种不同版本的XHR对象，即MSXML2.XMLHttp、MSXML2.XMLHttp3.3.0和MSXML2.XMLHttp.6.0。要使用MSXML库中的XHR对象，创建XHR对象时需要编写一个函数用于支持IE的早起版本。

2. XHR的用法

要使用XHR，第一个方法是open()方法,该方法不是真正发送请求，而是启动一个请求以备发送。它接受三个参数：请求类型（get、post等）、请求的URL和是否异步发送请求的布尔值。
